# PII Detection with DeBERTa (NLP)

This project implements a Named Entity Recognition (NER) model to detect Personally Identifiable Information (PII) in text using a DeBERTa transformer model.

The project was originally developed on Kaggle using the **Learning Agency Lab PII Data Detection dataset**.

## Project Overview

The goal is to automatically detect sensitive personal information in student writing, including:

- Names
- Email addresses
- Phone numbers
- Addresses
- Identification numbers

This helps protect privacy when sharing educational datasets.

## Model

The model uses:

- DeBERTa V3 transformer
- TensorFlow / Keras
- Token classification for NER

## Dataset

Learning Agency Lab – PII Data Detection

## Tools

- Python
- TensorFlow
- Keras NLP
- NumPy
- Pandas
- Plotly

## Results

The model achieved competitive performance on the Kaggle competition dataset.
